以下をそのまま貼り付けて使う。

```
あなたは実務レベルのWeb/UIデザイナー兼フロントエンド担当。宿のLP（1ページ）モックアップを**1回の出力で60〜80点**の完成度で作成してください。目的は、最終デザインに近い構成・見た目・コピーのたたき台を短時間で得ること。

# 対象
- 宿のLP（1ページ）
- 掲載先は**Airbnbのみ**（他OTA・予約サイトは載せない）
- Airbnb掲載URL（唯一の予約導線）: https://www.airbnb.jp/rooms/1557698987387425826

# 必須セクション/要素
- ヒーロー（宿の魅力コピー、CTA、代表写真）
- 予約ボタン（Airbnbへの遷移のみ）
- 料金の目安（不確実なら“例”として記載し、注記）
- アクセス情報（地名・交通手段は不確実ならプレースホルダ）
- 写真ギャラリー（複数枚）
- 滞在の特徴（静けさ、建築、自然、設備など）
- FAQ（3〜5件）
- 連絡/問い合わせ（Airbnb経由の案内）
- フッター（注意事項/規約の簡易記載）

# 重要ルール
- **外部リンクはAirbnbのみ**。他の予約/比較サイトは禁止。
- 不確実な情報は勝手に断定せず、**「要確認」注記**かプレースホルダで示す。
- デザインは“作り込み過ぎないが十分見える”レベル（60〜80点）。
- **シンプル過ぎるワイヤーではなく、配色・タイポ・余白感まで含めて示す**。
- 宿の世界観（木、暖色、静けさ、自然、伝統的要素）を想定し、トーンを合わせる。

# 出力形式
- **HTML + CSS + JS** で1ファイルを提示（1ファイル想定、`<style>` と `<script>` でOK）
- セクション順、見出し構造（h1→h2→h3）を守る
- 画像は仮でOKだが、**alt文は具体的に**（例: “赤瓦の屋根と木造の外観”）
- 予約ボタンはAirbnb URLへリンク

# 望ましいビジュアル方向
- 温かみのある木質・自然・伝統家屋の雰囲気
- 余白多め、落ち着いたトーン
- 色: 土色/木/生成り/深緑をベースに
- フォント: 日本語は読みやすく上品、英字は控えめ

# 期待するアウトプット
- 1回の出力で全体構成が揃い、**そのまま関係者に共有できる**レベル
- 具体的なコピーとUI要素（カード、アイコン、区切りなど）を含める

# 注意
- Airbnb以外の予約導線は載せない
- 誇張表現は避ける（事実不明な断定はNG）
```
